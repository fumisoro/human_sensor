#### 用途
人がある場所を通過したときに教えてほしい。  
->ホテルのフロント係の私。お客さんが来ない間はとっても暇なの！

2. HotelSonicwaveSensor下にあるJavaProjectをjarファイルにして実行する
3. Arduino、XBeeを接続して作成したjarファイルをUSBポートを選んで実行。
